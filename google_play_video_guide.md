# Инструкция для записи видео-демонстрации (Google Play Prominent Disclosure Video)

Для успешного прохождения проверки модераторами Google Play необходимо предоставить видеозапись, демонстрирующую процесс запроса фоновой геолокации.

## Пошаговый сценарий записи видео:

1. **Подготовка:**
   - Удалите приложение с тестового устройства (Android 11+) или сбросьте все ранее выданные разрешения в настройках приложения.
   - Включите запись экрана на устройстве.

2. **Запуск приложения:**
   - Откройте приложение Safe City. Пройдите авторизацию (если требуется).
   - Перейдите на главный экран (`HomeScreen`).

3. **Демонстрация Prominent Disclosure:**
   - Нажмите на кнопку экстренного вызова **SOS**.
   - На экране мгновенно появится диалоговое окно **«Разрешение на геолокацию»** (Prominent Disclosure) с текстом:
     > «Safe City собирает данные о местоположении для работы функции экстренного вызова SOS, даже когда приложение закрыто или не используется. Эти данные необходимы для оперативного прибытия службы охраны по вашим координатам».

4. **Первый этап запроса (Foreground):**
   - Нажмите кнопку **«Принять»** в диалоге.
   - Появится системное окно Android с запросом доступа к местоположению. Выберите **«При использовании приложения»** (While using the app).

5. **Второй этап запроса (Background / Allow all the time):**
   - Сразу после предоставления Foreground-доступа появится второе информационное окно **«Фоновый режим SOS»**, объясняющее необходимость выбора режима «В любом режиме».
   - Нажмите кнопку **«В настройки»**.
   - Откроется системный экран настроек разрешений приложения. Выберите **«Разрешить в любом режиме»** (Allow all the time).

6. **Проверка фоновой работы:**
   - Вернитесь в приложение — начнется процесс поиска и вызова охраны.
   - Сверните приложение в фоновый режим. В шторке уведомлений Android будет отображаться постоянное уведомление: **«Safe City SOS: Отправка координат охране в фоновом режиме»**.
   - Остановите запись экрана.

## Ссылка на видео
Загрузите полученное видео на YouTube (с доступом по ссылке) или Google Диск и укажите ссылку в консоли Google Play Console в разделе `App content -> Location permissions`.
